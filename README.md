# telescope-ghq.nvim

`telescope-ghq` provides its users with operating [x-motemen/ghq][].

[x-motemen/ghq]: https://github.com/x-motemen/ghq

## Installation

```lua
use{
  'nvim-telescope/telescope.nvim',
  requires = {
    'delphinus/telescope-ghq.nvim',
  },
  config = function()
    telescope.load_extension'ghq'
  end,
}
```
